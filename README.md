# Medical-Data-Complexity-Hybrid-Clustering
Exploring Medical Data Complexity: A Novel Hybrid Clustering Technique Using FCM and DBSCAN

Developed a medical data analysis system that combines Fuzzy C-Means (FCM) and DBSCAN clustering techniques to improve the identification of hidden patterns within complex healthcare datasets.

The project focused on preprocessing medical data, performing hybrid clustering analysis, and evaluating cluster quality to support better healthcare data interpretation. A user-friendly interface was designed to visualize clustering results and improve accessibility for users.

Technologies Used:
Python, Machine Learning, Data Mining, Clustering Algorithms, HTML, CSS, JavaScript

Skills Demonstrated:
Data Analysis, Machine Learning, Data Preprocessing, Clustering Techniques, Research Methodology, Problem Solving, Data Visualization
